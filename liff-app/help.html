วิธีใช้งาน

ภาพรวมระบบ

ระบบนี้ช่วยจำแทนคุณว่าขวดไหนถึงรอบถ่ายโอนเมื่อไหร่ ต้องใช้สูตรอาหารอะไร ใช้งานหลักๆ 3 อย่าง:

1. บันทึกถ่ายโอนเนื้อเยื่อ → 2. ดูสรุปข้อมูล/ประวัติย้อนหลัง → 3. รับแจ้งเตือนทาง Line อัตโนมัติ

ขั้นตอนบันทึกการถ่ายโอน

จากหน้าหลัก แตะชนิดพืชที่จะบันทึก แล้วกรอกทีละขั้น:

1. เลือกหมายเลขล็อต — พิมพ์เลขล็อตใหม่ หรือเลือกล็อตที่กำลังติดตามอยู่
2. จำนวนต้นที่ทำได้
3. ประเภทรอบนี้ TIB (อาหารเหลว) หรือ SS (กึ่งแข็ง) — ถ้าเลือก TIB ต้องกรอกหมายเลขขวดด้วย
4. ชั้นเก็บ + ช่องเก็บของรอบนี้ แล้วกด "บันทึก"

ระยะ (ขยาย / กระตุ้นราก / พร้อมออกปลูก), รอบถัดไป, และสูตรอาหารที่ต้องใช้ ระบบคำนวณให้อัตโนมัติทุกครั้งที่บันทึก ไม่ต้องจำเอง

Lab, Admin และ User

ทีมงานแบ่งเป็น 3 ห้อง Lab: KA / NJ / CC — ข้อมูลการถ่ายโอนของแต่ละ Lab แยกจากกันสมบูรณ์

Admin เห็น/จัดการข้อมูลได้ทุก Lab และตั้งสิทธิ์ผู้ใช้คนอื่นได้
User เห็นเฉพาะข้อมูลของ Lab ตัวเอง

เปิดแอปครั้งแรกผ่าน Line: ต้องรอแอดมินอนุมัติก่อนเสมอ (ขึ้นข้อความ "รอแอดมินอนุมัติ") หลังอนุมัติแล้วจะมีปุ่มให้เลือก Lab ของตัวเอง — เลือกได้ครั้งเดียว ถ้าเลือกผิดต้องแจ้งแอดมินให้เปลี่ยนให้ที่หน้า "ตั้งค่า" → "สิทธิ์ผู้ใช้"

ดูสรุปข้อมูลและประวัติ

ปุ่ม "📊 ดูสรุปข้อมูล" (Dashboard) — ดูจำนวนล็อตทั้งหมด ล็อตที่ใกล้ถึงกำหนด ภาพรวมตามชนิดพืชและที่เก็บ Admin เลือกดูเฉพาะ Lab เดียวหรือรวมทุก Lab ได้ และดูได้ว่าใครอยู่ Lab ไหนบ้าง

ปุ่ม "📜 ดูประวัติการถ่ายโอน" — ไล่ดูทุกรอบของล็อตที่เลือก ตั้งแต่รอบแรกจนถึงปัจจุบัน พร้อมวันที่ ตำแหน่งเก็บ และผู้บันทึกของแต่ละรอบ

การแจ้งเตือนทาง Line

บันทึกสำเร็จ → ได้รับแจ้งเตือนทันที (ผู้บันทึกเอง + แอดมินทุกคน)

ทุกเช้า → ระบบสรุปล็อตที่ถึงกำหนดถ่ายโอนวันนี้/พรุ่งนี้ส่งให้อัตโนมัติ แยกตาม Lab (แอดมินได้สรุปรวมทุก Lab)
